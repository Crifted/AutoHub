# Figma Designs – AutoHub

Alle UI-ontwerpen voor AutoHub zijn gemaakt in Figma.

## Link naar het ontwerp

[AutoHub – Figma Design](https://www.figma.com/design/NnYgBH9U5CiOcIwloXS4TZ/figma-designs-projecten-eig?node-id=11-411&t=Rvp8N5n7EqryFDpL-1)

---

## Wat staat erin

| Pagina | Omschrijving |
|---|---|
| Homepage | Hero-sectie, uitgelichte auto's, merkenrij, proefrit-CTA |
| Aanbod | Overzichtspagina met filters, zoekbalk en auto-kaartjes |
| Detailpagina | Galerij, specificaties, tuning-calculator, reviews, proefrit |
| Vergelijken | Tabel met auto's naast elkaar |
| Favorieten | Opgeslagen auto's van de ingelogde gebruiker |
| Login / Register | Authenticatiepagina's |
| Admin dashboard | Beheer auto's, proefritten, reviews, gebruikers |

---

## Design keuzes

- **Kleurpalet:** zwart canvas (`#08090B`) met platinum accent (`#D8DEE9`) en warm oranje als signaalkleur (`#E8A87C`)
- **Typography:** Sora (titels/prijzen) + Inter (bodytekst)
- **Stijl:** donker, luxe, filmisch — aansluitend op een premium automarkt
- **Thema:** licht/donker toggle beschikbaar
