## Samenvatting

AutoHub is een webapplicatie voor het bekijken, filteren, vergelijken en beoordelen van occasions. Het project is gebouwd als BPV examenopdracht voor de opleiding Software Development.

**Technische stack:** Node.js · Express · MySQL · EJS · HTML5/CSS3 · JavaScript

**Periode:** schooljaar 2025–2026
